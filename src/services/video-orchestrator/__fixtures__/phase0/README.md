# 一句话成片 Phase 0 回归快照

本目录保存阶段0使用的完全合成回归项目，不是生产数据库导出。

- 所有媒体 URL 使用不可联网的 `fixture://` 协议。
- 不包含用户 ID、邮箱、手机号、密钥、Cookie、Token 或 OSS 签名参数。
- 六个项目分别覆盖游戏广告、产品广告、电商广告、餐饮广告、剧情短片，以及多场景多机位短片。
- 每个项目都保留原始输入、planJson、front/side/back、边界关键帧、motion checkpoint 图片、视频片段、最终成片、调试产物，以及当前剧情失败点记录。

这些快照用于冻结结构和行为契约，不用于比较真实生成媒体的像素。真实视觉质量回归应使用独立、授权且去标识化的测试素材库。

更新规则：

1. 不得用生产项目直接替换本目录文件。
2. 新字段使用合成内容和 `fixture://` URL。
3. 更新 `baseline-manifest.json` 中的版本时，同时说明升级原因。
4. 任何变更都必须通过 `npm run test:one-prompt-video`。
